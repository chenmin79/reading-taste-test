# Example Usage

## Quick Start

### For AI Assistants

1. Copy the content from `reading-taste-test.md`
2. Use it as a system prompt or instruction
3. Guide users through the interview process

### Example Conversation Flow

**AI**: 我可以用一个多轮的小测试来判断你的阅读口味，不只是按类型分，还会看你偏好的密度、结构、目标和你真实喜欢/弃读过的书。你想要：
1. 简版（3–5分钟）
2. 标准版（更准）
3. 深度版（最准确）

**User**: 标准版

**AI**: 好，标准版开始。先做第一轮定位，你可以直接按题号回答：
1. 你最近更想读：小说 / 历史 / 社科 / 哲学 / 科普 / 传记 / 其他
2. 你读书主要为了：消遣 / 获取知识 / 建立框架 / 开阔视野 / 工作实用 / 情感共鸣
3. 你能接受的难度：轻松 / 中等 / 偏难 / 很难但值得
4. 你更喜欢：叙事推进 / 观点分析 / 案例解释 / 体系化梳理
5. 你更常避开什么：空泛 / 说教 / 术语堆砌 / 太浅 / 太学术 / 太啰嗦

**User**: 1. 科普，2. 获取知识，3. 中等，4. 案例解释，5. 空泛、说教

**AI**: 第二轮用你真实读过的书来校准，请尽量具体：
1. 说 3 本你很喜欢的书
2. 说 3 本你觉得还行但没那么喜欢的书
3. 说 3 本你不喜欢、弃读或觉得无聊的书
4. 每类里任选 1–2 本，简单说一句为什么

**User**: 
喜欢：《人类简史》《思考，快与慢》《枪炮、病菌与钢铁》
一般：《未来简史》《原则》《黑天鹅》
不喜欢：《如何阅读一本书》《精进》《自控力》

喜欢的：大框架，能让人重新看世界
一般的：有些重复，或者不够深
不喜欢的：太鸡汤，或者太工具化

**AI**: 最后做几道精细区分题：
1. 你更偏向：大框架解释 / 具体细节与文本
2. 你更偏向：新观点 / 经典定评
3. 你更偏向：人的故事 / 制度与结构
4. 你更偏向：通识综述 / 单点深挖
5. 你更偏向：清晰直白 / 风格浓、可反复咀嚼

**User**: 1. 大框架，2. 新观点，3. 制度与结构，4. 单点深挖，5. 清晰直白

**AI**: 
## 你的阅读画像

你是一个**知识密集型框架构建者**。你读书不是为了消遣，而是为了建立新的认知框架，重新理解世界。你特别喜欢那些能提供大视角、系统解释的书，但要求写得清晰、有案例支撑，不能空泛或说教。

### 关键维度
- 内容倾向：非虚构为主，科普/社科/历史
- 认知风格：框架驱动，喜欢系统性解释
- 难度容忍度：中等偏上，但要求清晰
- 结构偏好：案例解释，不喜欢纯理论或纯叙事
- 阅读目标：知识获取，建立思维模型
- 选书方式：主题驱动，不盲从评分

### 可能的雷区
- 空泛鸡汤类（如《精进》）
- 过度工具化、缺乏深度的畅销书
- 纯理论堆砌、缺乏案例的学术书

### 推荐清单

**Core Fit**（很可能会喜欢）
- 《人类的明天》- 尤瓦尔·赫拉利
  - 和《人类简史》同作者，大框架，清晰
- 《规模》- 杰弗里·韦斯特
  - 用物理学框架解释城市、生物、公司，系统性强
- 《复杂》- 梅拉妮·米歇尔
  - 复杂系统入门，案例丰富，框架清晰

**Strong Fit**（可能会喜欢）
- 《反脆弱》- 纳西姆·塔勒布
  - 大框架，但风格稍浓，需要耐心
- 《崩溃》- 贾雷德·戴蒙德
  - 《枪炮》作者，环境与文明，案例扎实
- 《理性选民的神话》- 布莱恩·卡普兰
  - 经济学框架解释政治，清晰有料

**Bridge Fit**（如果好奇可以尝试）
- 《哥德尔、艾舍尔、巴赫》- 侯世达
  - 大框架，但难度偏高，风格浓
- 《自私的基因》- 理查德·道金斯
  - 经典框架，但稍老，需要判断是否还新鲜

### 如果继续测，下一轮该问什么？
1. 你对"太难但值得"的书容忍度如何？（比如《哥德尔、艾舍尔、巴赫》）
2. 你更喜欢解释"为什么"还是"怎么办"的书？
3. 你对翻译作品的语言质量有多敏感？
4. 你有没有特别想深入了解的单一领域？
5. 你对"反直觉"观点的接受度如何？

## For Human Interviewers

### Standard Mode Questionnaire

**第一部分：基础偏好**
1. 你最常读哪些类型？
2. 你读书最主要是为了什么？
3. 你能接受多高的阅读难度？
4. 你更喜欢叙事、观点、案例还是框架？
5. 你最常放弃哪类书？

**第二部分：真实阅读记录**
6. 说 3 本你很喜欢的书。
7. 说 3 本你觉得一般的书。
8. 说 3 本你不喜欢/弃读的书。
9. 分别简单说下原因。

**第三部分：精细偏好**
10. 更喜欢大框架还是细节？
11. 更喜欢经典还是新书？
12. 更喜欢人的故事还是制度/结构？
13. 更喜欢综述还是深挖？
14. 更喜欢清晰直白还是风格浓烈？

## Tips for Best Results

1. **不要跳过真实书目** - 用实际读过的书校准最准确
2. **允许混合答案** - 用户可能有多种阅读模式
3. **区分"尊重"和"喜欢"** - 一本书被尊重不代表想要更多类似的
4. **区分"太难"和"太无聊"** - 两者都导致弃读，但推荐策略不同
5. **关注用户实际说了什么** - 不要只看类型标签
